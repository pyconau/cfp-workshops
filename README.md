# CFP Workshops

This repository contains a collection of resources for running talk writing workshops. The slides and presenters notes are coming 'soon'.

## What's included

- [`resources/event_template.md`](resources/event_template.md) - Template event description for promoting your workshop
- `resources/promo_template_*.png` - Promotional image templates
- Presentation slides for the workshop (coming soon)
- Presenters notes to facilitate each step of the workshop (coming soon)
- Train the trainer video, with guidance on how to facilitate a workshop (coming soon)

If you would like custom promotional graphics created for your event, please reach out to program@pycon.org.au ASAP.

## Promote your workshop

If you would like your workshop to be promoted by PyCon AU before 29 March 2026, please reach out to program@pycon.org.au.  Please include:
* The RSVP link to your workshop (e.g. meetup or luma)
* A link to your code of conduct (the PyCon AU code of conduct is open source)
* Any specific or offical names or text for us to include in a custom graphic for you.

We'll do our best to be responsive within a day or two (during February & March).
